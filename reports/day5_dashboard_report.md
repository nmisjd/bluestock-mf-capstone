# Day 5 Dashboard Report

## Project Title

**Mutual Fund Analytics Dashboard**

## Objective

The objective of Day 5 was to develop an interactive dashboard for analyzing mutual fund performance, investor behavior, and portfolio composition using Streamlit and Plotly. The dashboard enables users to explore fund-level insights through visualizations and filters.

---

## Technologies Used

* Python
* Pandas
* Streamlit
* Plotly
* SQLite
* Git & GitHub

---

## Dashboard Features

### 1. Overview Dashboard

The Overview section provides key performance indicators (KPIs) and high-level analytics.

**KPIs Displayed**

* Average 1-Year Return
* Average Sharpe Ratio
* Highest Assets Under Management (AUM)
* Total Funds Analyzed

**Visualizations**

* Risk vs Return Analysis
* Top 10 Performing Funds

---

### 2. Fund Analytics

The Fund Analytics section allows users to analyze individual mutual funds.

**Features**

* Fund Screener
* Fund Selection Dropdown
* Fund Performance Details
* Top Funds by Sharpe Ratio

**Insights**

* Comparison of risk-adjusted returns across schemes.
* Identification of top-performing funds based on Sharpe Ratio.

---

### 3. Investor Analytics

The Investor Analytics section provides insights into investor demographics and transaction patterns.

**Visualizations**

* Gender Distribution
* City Tier Distribution
* Investor Age Group Distribution
* Top States by Transactions

**Insights**

* Majority of investors belong to Tier-2 and Tier-3 cities.
* Investors aged 26–35 represent the largest segment.
* Significant transaction activity is concentrated in a few states.

---

### 4. Portfolio Analytics

The Portfolio Analytics section analyzes portfolio holdings and sector allocation.

**Visualizations**

* Sector Allocation Pie Chart
* Top Holdings by Market Value
* Holdings Summary Table

**Insights**

* Banking, IT, and Pharma sectors constitute a major portion of portfolio allocations.
* Large-cap stocks dominate portfolio holdings.
* Sector diversification can be evaluated visually.

---

## Business Value

The dashboard provides a centralized platform for:

* Evaluating mutual fund performance.
* Comparing risk and return metrics.
* Understanding investor demographics.
* Monitoring sector-wise portfolio exposure.
* Supporting investment decision-making through data-driven insights.
