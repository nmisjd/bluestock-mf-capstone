# Mutual Fund Analytics Platform

## Problem Statement
Analyze mutual fund performance, investor behavior, and portfolio composition using financial datasets.

## Solution
Built a complete analytics pipeline with data ingestion, cleaning, database creation, exploratory analysis, financial metrics, and an interactive dashboard.

## Key Features
- ETL Pipeline
- SQLite Database
- EDA
- CAGR Analysis
- Volatility Analysis
- Sharpe Ratio Analysis
- Investor Analytics
- Portfolio Analytics
- Interactive Dashboard

## Technologies
Python, Pandas, SQLite, Streamlit, Plotly, Git, GitHub

## Outcome
Developed a fintech analytics platform capable of generating actionable insights from mutual fund data.
